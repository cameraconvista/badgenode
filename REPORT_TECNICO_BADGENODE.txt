================================================================================
1. PANORAMICA GENERALE
================================================================================

BADGENODE è un sistema web-based per la gestione delle timbrature dipendenti che:
- Utilizza Supabase come backend PostgreSQL
- Frontend in JavaScript vanilla (NO React/Angular/Vue)
- Gestisce PIN numerici per identificazione dipendenti
- Calcola automaticamente ore lavorate, straordinari, giorni mancanti
- Supporta esportazione in multiple formati (PDF, Excel, WhatsApp, Google Sheets)
- Gestisce turni notturni con logica "giorno logico"

STACK TECNOLOGICO:
- Frontend: HTML5, CSS3, JavaScript ES6+ (moduli)
- Backend: Supabase (PostgreSQL + API REST)
- Build Tool: Vite.js per sviluppo
- Librerie: jsPDF, SheetJS, js2py per export
- Deploy: Netlify/Replit

VINCOLI E INDICI:

- Email unica
- Ore contrattuali default 8.00
- Indice su PIN per performance

2.2 TABELLA TIMBRATURE
----------------------
Struttura:

CREATE TABLE public.timbrature (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  pin INTEGER NOT NULL,
  nome TEXT NOT NULL,
  cognome TEXT NOT NULL,
  tipo TEXT NOT NULL CHECK (tipo IN ('entrata', 'uscita')),
  data DATE NOT NULL,
  ore TEXT NOT NULL,              -- Formato HH:MM:SS
  giornologico DATE,             -- Per gestione turni notturni
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  
  CONSTRAINT fk_timbrature_utenti 
    FOREIGN KEY (pin) REFERENCES utenti(pin) ON DELETE CASCADE
);

INDICI PER PERFORMANCE:
- idx_timbrature_pin
- idx_timbrature_data
- idx_timbrature_giornologico

RELAZIONI:
- Foreign Key CASCADE: eliminando utente si eliminano sue timbrature
- Denormalizzazione: nome/cognome ridondanti per performance query

4.1 LOGICA CORE (assets/scripts/storico-logic.js)
-------------------------------------------------
Algoritmo principale:

function calcolaOreLavorate(timbratureGiorno) {
  const entrate = timbratureGiorno.filter(t => t.tipo === "entrata")
                                  .sort((a, b) => a.ore.localeCompare(b.ore));
  const uscite = timbratureGiorno.filter(t => t.tipo === "uscita")
                                 .sort((a, b) => a.ore.localeCompare(b.ore));
  
  if (entrate.length > 0 && uscite.length > 0) {
    const primaEntrata = entrate[0];
    const ultimaUscita = uscite[uscite.length - 1];
    
    // Conversione a oggetti Date per calcolo
    const oraInizio = new Date(`1970-01-01T${primaEntrata.ore}`);
    const oraFine = new Date(`1970-01-01T${ultimaUscita.ore}`);
    
    // Gestione turni notturni
    if (oraFine < oraInizio) {
      oraFine.setDate(oraFine.getDate() + 1);
    }
    
    const diffMs = oraFine - oraInizio;
    return diffMs > 0 ? diffMs / (1000 * 60 * 60) : 0;
  }
  return 0;
}

6.1 FILTRI PREDEFINITI (assets/scripts/storico-logic.js)
--------------------------------------------------------
function aggiornaRange(mese) {
  const oggi = new Date();
  let anno = oggi.getFullYear();
  let meseIndice = oggi.getMonth();

  switch(mese) {
    case "corrente":
      // Mese corrente
      break;
    case "precedente":
      meseIndice -= 1;
      if (meseIndice < 0) {
        meseIndice = 11;
        anno -= 1;
      }
      break;
    case "due-precedenti":
      meseIndice -= 2;
      if (meseIndice < 0) {
        meseIndice += 12;
        anno -= 1;
      }
      break;
  }
  
  const primoGiorno = new Date(anno, meseIndice, 1, 12, 0, 0);
  const ultimoGiorno = new Date(anno, meseIndice + 1, 0, 12, 0, 0);
  
  dataInizio.value = primoGiorno.toISOString().split('T')[0];
  dataFine.value = ultimoGiorno.toISOString().split('T')[0];
}

9. GESTIONE TURNI NOTTURNI
================================================================================

9.1 CONCETTO "GIORNO LOGICO"
----------------------------
Problema: turni che iniziano un giorno e finiscono il successivo
Soluzione: campo "giornologico" che raggruppa timbrature per turno

Esempio:
- Entrata: 23:30 del 15/01 → giornologico = 15/01
- Uscita: 02:30 del 16/01 → giornologico = 15/01

Entrambe appartengono al turno del 15/01

9.2 CALCOLO GIORNO LOGICO (index.html)
--------------------------------------
function calcolaGiornoLogico(dataTimbratura, ora) {
  const [ore, minuti] = ora.split(":");
  
  if (parseInt(ore) >= 0 && parseInt(ore) < 5) {
    // Ore 00:00-04:59 → giorno precedente
    const dataGiornoLogico = new Date(dataTimbratura);
    dataGiornoLogico.setDate(dataGiornoLogico.getDate() - 1);
    return dataGiornoLogico.toISOString().split('T')[0];
  }
  
  // Ore 05:00-23:59 → stesso giorno
  return dataTimbratura;
}
